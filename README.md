# 测试Reactor模式

总共四种类型的Reactor方案：

1. 单 Reactor 单进程 / 线程；
2. 单 Reactor 多进程 / 线程；
3. 多 Reactor 单进程 / 线程；
4. 多 Reactor 多进程 / 线程。
